# Jogo da Velha

Para iniciar o servidor e criar novas salas, entre na pasta `<src>` e execute o arquivo `<server.py`:

  ```
  $ cd src
  $ python3 server.py
  ```
  
Para iniciar um novo cliente, entre na pasta `<src>` e execute o arquivo `<client.py`:

  ```
  $ cd src
  $ python3 client.py
  ```
